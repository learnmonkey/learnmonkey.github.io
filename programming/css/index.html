Introduction to CSS

What is CSS?

CSS is used to make HTML websites look better. Using CSS, you can set fonts, font-sizes, colors, opacity, and a lot, lot more.

How do I run CSS?

You can directly put it in your HTML files or create a new file for the CSS and reference it from the HTML.

Prerequisites

  • Knowledge of HTML
  • A web browser
