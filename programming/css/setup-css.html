Setting Up CSS

How do I setup CSS?

It is actually very easy to get CSS setup. To do that, firstly, make a file in the same directory as your index.html file called style.css. This is where all of our CSS will go. Then, in your index.html file, add the following code into your <head>:

<link href="style.css" rel="stylesheet" type="text/css" />

Checking If It Works

Once you have CSS added into your HTML, add the following code into the CSS file:

body { font-family: sans-serif; }
