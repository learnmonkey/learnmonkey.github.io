Introduction to Go

What is Go?

Go is a programming language created at Google in 2009. It is compiled, which means that Go is compiled by the Go compiler to binary ones and zeros that your computer can understand.

Why use Go?

Go is a popular programming language designed to be static-typed and have a quick run-time like C but also be readable like Python or Javascript.
