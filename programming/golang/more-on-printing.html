Printing In Go

What is Printing?

In the last tutorial when we made our first program, we wrote Hello, World! in the Terminal/Command Prompt. In programming, we call this printing. Printing is an essential but basic part of programming; It's just writing text in the console. If we wanted to create a text-based game, we would use print to show the text in the terminal.

How can I print text in Go?

We already know that we can simply write fmt.Println(text) to print stuff in Go. We can change text to be any Data Type we want (to learn more about Data Types, click the link to see our Data Types tutorial)!

Usage

There are many ways you can use the Go fmt.Println function

Firstly, you can simply put your text as a single string inside the function. In short, a string is simply a group of characters STRUNG together. In Go, we surround strings with double quotation marks (") and we also end them with double quotation marks

For example, some examples of doing this are:

fmt.Println("My name is Mr. Monkey!")
fmt.Println("suoguhgo")

We can also use fmt.Println to print a number: an integer or decimal(floating point). To do that, we can write fmt.Println(number).

For example, that could look like:

fmt.Println(36257265)
fmt.Println(973.256347)

Finally, we can place different Data Types in the function, seperated with commas like this:

fmt.Println("My", "Name", "Is", "Joe")

This will tell the fmt.Println function to print each of the strings but sperated with spaces so the end result will be: My Name Is Joe.
