Go While Loop

What is a While Loop?

A while loop like an if statement, but it runs as long as that condition is true, instead of just running it once.

Making a While Loop

Actually, the title of this tutorial is quite misleading since Go has no while loops. This tutorial was only created because we couldn't find a place to put it in the tutorial on for loops. Instead, Go uses the for loop to replace the while loop, which is a seperate loop in some other programming languages. Below is the syntax: 


for condition {
    // code to keep repeating
}

Something like the below would run indefintely: 


for true {
    // the code inside the loop runs forever
}
