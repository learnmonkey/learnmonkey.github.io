Printing In Go

What is Printing?

In the last tutorial when we made our first program, we wrote Hello, World! in the Terminal/Command Prompt. In programming, we call this printing. Printing is an essential but basic part of programming; It's just writing text in the console. If we wanted to create a text-based game, we would use print to show the text in the terminal.
