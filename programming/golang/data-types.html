Data Types In Go

What is a Data Type?

A data type is a type of data that we can assign to a variable. In Go, we have a few basic data types that we can put together to make complex ones. In this tutorial, we'll be covering the most important ones.

String

A string is some text. If that's hard to remember, think of text as characters strung together on a string. In Go, strings are wrapped in double quotes.

Integer

An integer is a whole number that can be positive or negative. In Go, we simply write out the number, no quotes or anything like that. To make it negative, just add a dash in front of it.

Boolean

Booleans are either true or false. Since computers are made from ones and zeroes, which are equivalent to true and false, you'd imagine booleans are pretty important. In Go, to make a variable true, we simply type true, all lowercase. Liekwise, we type false for false. Booleans are named after the English mathematician George Boole.

Floats

Floats are an abbreviation of floating point numbers. Floats are simply decimals. In Go, there are two types of floats, float32 and float64. float64 uses 64 bytes of memory, while float32 only uses 32 bytes of memory. So, float64 can store larger floats and floats with more decimals. However, float64 uses more memory, so there is a tradeoff.
