JavaScript Alerts

What is an Alert?

In JavaScript, an alert is a little message box on the webpage. Alerts are great for testing JavaScript code since you do need to go to the console, but alerts should generally be avoided when in a production environment.

Making an Alert

To make alerts, we use the window.alert() function. You can run it from the console or a JavaScript file. For example, type the following code: 


window.alert("Hello World!")
