Setting Up jQuery

How to Setup jQuery

jQuery is hosted on online CDNs. For this tutorial, we will include the jQuery library using the Cloudflare CDN. To do this, simply add


<script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/3.6.0/jquery.min.js"></script>

To your head tag. So, your head might look like this:


<head>
<script src="script.js"></script>
<script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/3.6.0/jquery.min.js"></script>
</head>
If your script are NOT in your head, than you MUST put the jQuery script tag before the regular script.js tag, or your code will work. Ignore this if your JavaScript scripts are in your head.

Now, you are ready to write your jQuery code!
