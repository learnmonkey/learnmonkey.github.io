Python Functions

What is a Function?

A function is similar to a variable, but instead of storing numbers or text, it stores lines of code for Python to run. Just like we learned in math class, functions in Python can have inputs.

How to Make a Function?

To make a function, we use def, which stands for "define". After that, we type our function name, which in this case is helloWorld and type a pair of parentheses (()). After that, we type a colon and press enter. Then, Sublime Text should indent the next line. This is because Python uses indentation for seperating blocks of code and this includes functions.

Note: You cannot put a space in the middle of a function name.

Then, inside of our function we can type print("Hello World!"). This tells Python that whenever we run the helloWorld function, we want to print Hello World!.

Function Arguments and Parameters

When we are making our function, we can make our function have specific inputs so that when we use our function. These are called arguments. Then, when we run our functions, we can pass in the arguments we just defined as parameters. We can do it as follows:


def helloWorld(argument1, argument2, argument3, .....):
    # do something

We put the arguments in the parentheses and seperate them by commas. Then, we can use the arguments as if they were variables. If we put nothing in the parentheses, that tells Python that we're passing in no arguments.

Running a Function

Running a function quite similar to making a function, but we don't use def and we don't type anything after the parentheses. For example, we could call the helloWorld fucntion we made earlier by typing helloWorld().
