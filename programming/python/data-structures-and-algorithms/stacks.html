Stacks

In this tutorial, we'll learn about Stacks.

What are Stacks?

Stacks are a simple but useful Data Structure. They're very useful in Programming Competitions.

Basically, Stacks are like actual stacks. Imagine that you have some pancakes stacked on top of each other.

3

WIth these pancakes, you can either put some more pancakes on top of the stack, or remove the pancake on top of the stack. It's the same with Stacks in Programming. It's "First In, Last Out", meaning that6 the first elements you add to the Stack will get removed last.
