In this tutorial, we'll learn about Binary Search, a simple and speedy Search Algorithm.

Binary Search Implementation

Due to its simplicity, a Binary Search is very easy to implement. We only need to iterate, or loop, over a list of elements and check if any of them are the same as the element we are looking for. Below is the implementation of a Binary Search:


def Binary_search(arr, x):
    """Finds the index of x in arr using a Binary Search Algorithm"""
    for index in range(len(arr)):
        if arr[index] == x:
            return index
    return -1

Running the Code

The code is written in a simple function. It performs a Binary Search on the array, arr, and finds the first occurence of the value of x. If the element is not found, the program returns -1. Otherwise, it returns the index where x is found.
