Troubleshooting PIP

If pip install [package] doesn't work for you, try the following commands:

  • python -m pip install [package]
  • py -m pip install [package]
  • python3 -m pip install [package]
You need to have Python added to your Path in order to use PIP. If you don't, then reinstall Python but make sure to add it to your "Path". You can select that option in the Wizard Setup.
