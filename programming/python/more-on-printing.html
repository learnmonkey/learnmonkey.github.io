More on Printing

In the last tutorial, we learned that we could print stuff into the command line by using the word print. In this tutorial, we'll learn a bit more about printing.

Printing Multiple Things at a Time

We can give print two things to print by separating them by commas.

For example, we can do print("Hello", "World!"). In the command line, it prints out Hello World!.

If you noticed, Python put a space between the words.
