Python Trivia Game

In this tutorial, we'll create a simple command line interface (CLI) program that runs a quick game of trivia.

Making Questions

For our trivia game, we'll need to make some questions and store them for our program to access. We'll have them in this format:


questions = [{"question" : "Who created Python?", "possible" : ["Guido Van Rossum", "James Gosling", "Donald Knuth", "Dennis Ritchie"], "correct" : "Guido Van Rossum"}]
