Discord.py Bot

The Problem

If we wanted to make a real Discord bot with commands and a prefix, we could do lots and lots and lots of elif statements in the message listener, but that would be pretty inefficient and quite ugly. Instead, we could've used Discord.py's built in Bot object and set a prefix and make commands much more easily.
