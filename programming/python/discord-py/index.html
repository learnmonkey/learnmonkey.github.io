Introduction to Discord.py

What is Discord.py?

Discord.py is an easy-to-use Python module for creating Discord bots. Currently, there are two versions of Discord.py. The new one is called Discord.py rewrite, and it will be the one that will be used.

Features

Practically everything that Discord has to offer can be used with Discord.py. Some features include:

  • Sending and Receiving Message
  • Logging Activity
  • Creating Embeds
  • Server Moderation
  • Playing Sounds
  • And much more!
