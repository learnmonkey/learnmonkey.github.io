Python Environment Setup

Now, it's time to set up our environment.

Installing Python

First of all, we need to install Python. So, go to python.org/downloads. Then, click the big yellow button:

After that, the Python installer should be installed on your computer. Simply run the installer and go through all the steps. For a more detailed explanation of the Python installation process, click here. After that, Python will be installed. To check if Python has been installed or not, simply type in "Python" in the Windows search bar and see if Python comes up.

Installing Sublime Text

We need some place to run and write our code. So, we will be using Sublime Text, which is a easy-to-use code editor where we can easily run our Python code.
