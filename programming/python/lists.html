Python Lists

What are Lists?

Lists in Python are basically just lists of elements. The elements can be anything: strings, numbers, you name it. You can add and remove elements from lists.

How to Make Lists?

It's quite simple. We just seperate our elements with commas and wrap the entire thing in square brackets ([ and ]), which tells Python that we're making a list. We can put whatever we want in there.

Getting and Setting a Value

Getting an element in a list is easy. You do it like this: list[index]. You replace list with the name of the list variable of index with the index of the element in the list. Indexes start at 0 for the first element and keep going up by one.

Setting a value is almost exactly the same as setting one. We simply set the element to something. So, we do list[index] = value. It's exactly the same as last time, but we replace value with the value we want to set.
