Python Lists

What are Lists?

Lists in Python are basically just lists of elements. The elements can be literally anything: strings, numbers, you name it. You can add and remove elements from lists.

How to Make Lists?

It's quite simple. We just seperate our elements with commas and wrap the entire thing in square brackets ([ and ]), which tells Python that we're making a list.
