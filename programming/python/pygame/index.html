Introduction to Pygame

What is Pygame?

Pygame is a simple-to-use Python module for 2D game development. It is the most used Python game development framework.

Why should I use Pygame?

Since Pygame is so easy to set-up and start using, it's a great intro to simple Game Development in Python. Do note, however, it isn't that most advanced so if you want to development a really complex game, try using a Game Engine like Unity.

Prerequisites

  • Experience with Python
