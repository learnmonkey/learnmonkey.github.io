Routing

What is Routing?

Routing simply means returning code (or HTML pages) when someone visits a specific URL on your site. For example, in the previous tutorial, we had one route: the / route which was our home page. In this tutorial, we will learn a little bit more about routing with Flask.

Making Routes In Flask

Flask is what we call a Backend Web Framework, or a framework for creating the backend code for a site. Backend is handling the user requests and the code on the server. Frontend programming is creating the actual user interface, or the HTML code, on the site. Anyways, how do you make routes in Flask? Well, we simply type 

@app.route("/my-route")
and then followed by a function. In that function, we will return HTML code as a string. You can also return HTML code as a template, as we will learn in the next tutorial.
