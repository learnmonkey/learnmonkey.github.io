Python F-Strings

F-strings were introduced in Python 3.6 as an improved way of formatting strings. Before f-strings, we would either have to concatenate a bunch of strings or use the format() method. Both of them aren't particularly elegant. So, f-strings were introduced. Using them is similar to using the format() method, but everything is more concise. For example, instead of writing: 


name = "John"
age = 36
txt = "Hello! My name is {name}. I'm {age} years old.".format(name = name, age = age)

We could write: 


name = "John"
age = 36
txt = f"Hello! My name is {name}. I'm {age} years old."
