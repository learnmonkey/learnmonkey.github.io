Introduction to Bottle

What is Bottle?

Bottle is a super lightweight microframework for creating web applications with Python. It's super easy to get started. It, however, doesn't have a lot of bigger functions like Django so it is more recommended for beginners.

Bottle is a backend web framework which means that Bottle runs the code behind the scenes. Backend means that Bottle handles all the user requests and the code on the server.

Fun fact! Did you know that Bottle doesn't have any dependencies other than the Python standard library? It's true!!

Prerequisites

  • Experience with Python
  • Web Browser
