Your First Python Program

Now, it's time for you to write your first program.

Writing The Program

Our very first program will simply print out "Hello World!" onto the screen. That might soudn boring, but don't worry, in later lessons, you'll learn more advanced things.

So, first, open up a text editor and make a file ending in .py. ".py" is the file extension for Python, which means that all of the file names for Python programs end in .py.

You can name your file whatever you want, but in this tutorial, I'm calling it "helloworld.py".

The program only has one line: print("Hello World!")

Finally, save the file.

Running The Program

Go into the terminal and navigate to the folder where you created your Python program.

Then, execute the following command:

python [filename].py

This tells Python to run the Python program.

After running the command, you should see the words "Hello World!" printed below your command.

Yay! You've written and run your first Python program.
