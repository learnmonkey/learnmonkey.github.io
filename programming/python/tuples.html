Python Tuples

What are Tuples?

Tuples are the same as lists in most aspects. For example, you can put whatever you want in tuples. However, you cannot change the tuple whatsoever once you make it.

How to Make Lists?

It's the same as lists, but we use parentheses (( and )) instead of square brackets. We seperate the items in the tuple by commas.

Getting an Element

Getting an element is the exact same in tuples as in lists. We do tuple[index], where tuple is the tuple variable and index is the index of the element. Indexes start at zero for the first element and go up by one for each element.
