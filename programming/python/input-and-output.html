Input and Output

What is Input and Output?

In programming, input and output means communication between the computer and the human. This can take a number of forms.

One example is a submit button on a form. When you click the button, you are communicating something with the computer, namely that you are submitting the form. This is input

Once you submit the form, the computer responds with a success message. The computer is communicating with you.

You will often see input and output abbreviated as input/output or simply I/O.

In this tutorial, input and output will mean you entering something in the command line and the computer printing something back.

Input

Input in Python is incredibly simple. You just do input().
