What Is Python?

Python is a simple but powerful programming language. Python is one of the most popular programming languages and can be used to create everything from websites to artificial intelligence.

History

Guido van Rossum, the creator of Python, first started working on Python in the 1980s. Python was first released in 1991. Currently, Python is in version 3.

Why Python?

Python is simple and easy to pick up but it is also very powerful, being able to make everything from websites, games, and even to artificial intelligence. Plus, Python is used widely in real-life and learning Python is great for your resume.

Many other people also use Python and have made things called modules, which are basically reusable bits of code. Since Python has such as large community, there are modules for everything from computer vision to desktop applications. Moreover, Python also has a lot of built-in modules.

Uses

Python is a general-purpose programming language, so it can be used to TONS of different things. Some uses of Python include:

  • Websites
  • Dekstop Applications
  • Game Development
  • Data Science
  • Artificial Intelligence
  • Scientific Computing
  • Basically anything else you can imagine!
