Your First Jekyll Website

Creating a Jekyll Website

To generate all of the Jekyll boilerplate stuff, we can use some command line commands.

  1. Install all prerequisites.
  2. Install the Jekyll and bundler gems: 
    3. gem install jekyll bundler
  3. Create a Jekyll website with all the boilerplate (will be created in ./mysite): 
    4. jekyll new mysite
  4. Go into the mysite directory: 
    5. cd mysite
  5. Add webrick to the bundle: 
    6. bundle add webrick
  6. Build the site and run it on localhost: 
    7. bundle exec jekyll serve
  7. Go to https://localhost:4000 in your web browser.
