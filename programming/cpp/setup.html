C++ Setup

Before we can do anything with C++, we should have a compiler and a code editor. Luckily, the Code::Blocks IDE has both of those taken care of for us. So, go to Code::Blocks's website and navigate to the Doanloads page. There, go to the binary releases page and install the codeblocks-20.03mingw-setup.exe file.
