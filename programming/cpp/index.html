Introduction to C++

What is C++?

C++ is a low-level programming language which means that you can do things like directly manipulate memory. C++ is based on the C programming language, except C++ has classes and other things.

History

C++ is older than many other programming languages, being first released in 1985. C++ was created by Bjarne Stroustrup as the successor to C.

Uses

C++ is very fast, so it is used in places where speed and efficiency are very important:

  • Games
  • Operating Systems
  • Animation
  • Databases
  • Compilers
  • And much more!

How to Run C++?

C++ is a compiled language which means that C++ takes your code and turns it into an executable, which is machine code that your computer can natively understand, no compiler or interpreter needed. Then, we run the executable.
