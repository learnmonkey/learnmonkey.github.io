Translations

What are Translations?

A Translation is a Transformation in which you move a shape left, up, right, down, or diagonally, but doing the same Translation to each point on the shape. You MUST not mess up or forget any points, or the Translation will not be correct.

Examples

Here are a few examples of Translations:

Example #1

Translation Right By 5 Units













In this example, we translate the square 5 units right, meaning that we move it without messing with how the shape looks.

Example #2

Translation Down By 10 Units



In this example, we translate the triangle 10 units down, meaning that we move it down without messing with how the shape looks so the triangle still looks the same.

Example #3

Translation Diagonally 10 Units









In this example, we translate the triangle 10 units down, meaning that we move it down without messing with how the shape looks so the triangle still looks the same.
