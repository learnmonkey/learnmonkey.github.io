Translations

What are Translations?

A Translation is a Transformation in which you move a shape left, up, right, down, or diagonally, but doing the same Translation to each point on the shape. You MUST not mess up or forget any points, or the Translation will not be correct.

Different Translations

There are three main transformations:

  • The Translation
  • The Reflection
  • The Rotation

Of course, there are also some other Translations like Dilations but these are the main ones.
